Классификатор животных на Keras

Нейросеть для распознавания изображений животных с использованием TensorFlow/Keras.

Этот проект представляет собой нейронную сеть, написанную на Python с использованием TensorFlow/Keras, которая умеет классифицировать изображения трёх животных:

🦫 Бобр
🐀 Крыса
🐹 Хомяк

Модель обучается на собственном наборе изображений и после обучения может определять, какое животное изображено на фотографии.

🧠 Обучение нейронной сети на собственном датасете
🖼️ Классификация изображений животных
⚡ Быстрое тестирование обученной модели
💾 Сохранение и загрузка модели
🐍 Полностью написано на Python

⚙️ Установка

Установите все необходимые библиотеки командой:

pip install -r requirements.txt
🚀 Обучение модели

Запустите файл:

python enymalmodel.py

Во время выполнения программа:

📂 Загружает изображения из датасета.
🧠 Создаёт нейронную сеть.
📈 Обучает модель.
💾 Сохраняет готовую модель для дальнейшего использования.
🔍 Тестирование модели

После обучения можно проверить работу модели:

python anymalmodel_test.py

Программа загрузит сохранённую модель и определит, какое животное изображено на выбранной фотографии.
